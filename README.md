### A Pygame Game

An exploration ARPG with fantasy D&D elements.

Uses Tiled level designer.
https://www.mapeditor.org/

> * python 3.13+
> * pygame 2.0.1
> * PyTMX 3.30

## Usage

Make sure the packages are installed.
```bash
uv sync
```

and simply run the main file.

```bash
uv run main.py
```
